# Django-Social-Auth

Steps to do after recieving the files

Run the following commands in your server:
1) sudo pip install django-allauth
2) go inside the project folder and run sudo python manage.py migrate

1) Login to your facebook developer account.
2) Create a app 
3)copy the app id and the app secret
4)After that click on settings on the left side of the page
5) Below in the website portion write http://localhost:8000
6) then go to localhost:8000/admin
7)login in as a root user.
8)click on social applications
9) click on example_facebook replace the clientid with app if and secret with secret that you have obtained form your developer accounts
